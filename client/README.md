# Vite React Frontend

Quick start guide for development and production.

## Setup

```bash
git clone <repo-url>
cd <project-folder>
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Build & Preview

```bash
npm run build
npm run preview
```

Production-ready files are in the `dist/` folder.
